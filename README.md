## **Инструкция по запуску программы:**
Версия Java - 17

Система сборки gradle, версия 8.8

Из сторонних библиотек использовался только lombok:

    (Зависимости для gradle)    
    compileOnly 'org.projectlombok:lombok:1.18.30'
    annotationProcessor 'org.projectlombok:lombok:1.18.30'

**Команда для запуска jar файла:** java -jar SHIFT_test_file_parser-1.jar -f -p result_ in1.txt in2.txt

**Если запуск производится не из директории проекта то:**

java -jar [путь\до\файла\SHIFT_test_file_parser-1.jar] -f -p result_ [путь\до\входного\файла]

По умолчанию результат будет сохранен в ту директорию, из которой был произведен запуск.

Аргументы:

    -o [ваш путь вывода] - путь, по которому будут созданы выходные файлы
    -p [ваш префикс] - префикс перед именем файлов вывода
    -a - режим добавления результата в уже существующие файлы вывода
    -s - режим отображения краткой статистики
    -f - режим отображения полной статистики, имеет приоритет над -s
    порядок не важен, путь вывода нужно указать через пробел сразу после -o, как и префикс после -p.
    Все, что не начинается с '-' и не идет сразу после -s или -p расценивается как имя входного файла. 
    Также можно указать путь до файла, если он находится в другой папке
